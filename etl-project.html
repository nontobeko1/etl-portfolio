Automated Retail Sales ETL Pipeline

Overview

This project implements an automated ETL pipeline that extracts raw retail transaction data, applies data quality and transformation rules, and loads the data into a SQL Server star schema for analytics.

Architecture

  • Raw CSV source data
  • Python-based extract and transform
  • Staging table for safe loading
  • Star schema (fact + dimensions)
  • Automated pipeline execution

Technologies Used

  • Python (pandas, pyodbc, SQLAlchemy)
  • Microsoft SQL Server
  • Star Schema Modelling
  • ETL Automation

Key Features

  • Data cleansing and validation
  • Bulk loading for performance
  • Dimension upserts using MERGE
  • Idempotent pipeline execution
