About Me

I’m a data professional who enjoys turning messy, raw datasets into clean, trusted data that teams can use for reporting and decision-making. I work across the full pipeline — extracting data, transforming it with strong validation rules, and loading it into well-structured warehouse models like star schemas.

My recent work includes building an end-to-end Online Retail ETL Pipeline using Python (Pandas + SQLAlchemy/pyodbc) and Microsoft SQL Server. The pipeline loads data into a staging layer, builds dimensions, and refreshes a fact table designed for analytics and BI.

I’m comfortable working in cross-functional environments, documenting solutions clearly, and iterating fast to fix issues. I’m actively growing my skills in cloud data engineering and modern data tooling.
